This repo contains two variants that both run fully in your browser. When served over localhost, each variant gets its own origin-scoped storage.

TUSK-OS (SQLite/sql.js)

SQLite engine in WebAssembly (sql.js) with a local JSON fallback. Export/import a real .sqlite file.

TUSK-OS (IndexedDB)

Native IndexedDB storage with export to JSON, CSV, and .sqlite (via sql.js in-memory export).
